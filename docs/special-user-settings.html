スペシャルユーザー設定 取扱説明書

特定のニコニコユーザーを登録し、その人がコメントした放送の保存、AI分析、HTML生成、自動反応を配信者ごとに設定する画面の説明書です。操作はすべてGUIから行えます。

スペシャルユーザーを登録する

  1. メインGUIで「スペシャルユーザー」タブを開きます。
  2. 「スペシャルユーザーID または user URL」に数値IDかニコニコのユーザーページURLを入力します。
  3. 必要なら「名前」と「メモ」を入力します。
  4. 「登録/更新」を押します。

名前を空欄にして数値IDを登録した場合は、取得できるときだけユーザー名が自動入力されます。同じユーザーIDをもう一度登録すると、その登録内容を更新します。

名前GUIや生成HTMLで使用する表示名です。
メモ管理用の自由記入欄です。
更新日時最後に設定を保存した日時です。
  • 詳しい設定を開くには、一覧の対象行をダブルクリックします。
  • 削除するには行を選択し、「選択削除」を押します。
  • 削除すると、そのユーザーと配信者の紐付けやトリガー設定も削除されます。

左側でAIと反応方法を設定し、右側で検出対象となる配信者を設定します。最後に画面下部の「保存」を押すまで変更は確定しません。

紐づいている配信者一覧

スペシャルユーザーを、どの配信者の放送で探すかを指定します。

  1. 「配信者ID / user URL / ch URL」に配信者ID、ユーザーURL、またはチャンネルURLを入力します。
  2. 名前が空欄なら「名前取得」を押します。
  3. 「配信者追加」を押します。
  4. 不要な配信者は一覧で選び、「選択削除」を押します。

数値ユーザーIDのスペシャルユーザーでは、「フォロー中を一括登録」を押すと、そのユーザーがフォローしている配信者をまとめて追加できます。

配信者を追加しただけでは、編集画面全体の保存は完了していません。最後に下部の「保存」を押してください。

保存と有効化

  1. 一覧画面でスペシャルユーザーの「有効」にチェックします。
  2. 編集画面で対象配信者を追加し、その配信者も有効にします。
  3. 必要なAI設定、反応設定、サーバー設定を入力します。
  4. 画面下部の「保存」を押します。

「キャンセル」を押すと、編集画面で行った未保存の変更は破棄されます。既に保存済みの放送HTMLは、設定を変えただけでは作り直されません。

最初におすすめする設定

  1. ユーザーIDを登録して「有効」をオンにします。
  2. 対象配信者を1人だけ追加して動作を確認します。
  3. 最初は最大反応回数を1にします。
  4. 自動投稿が不要なら基本反応設定とHTMLサーバー送信をOFFにします。
  5. 保存後、確認タブとログで検出結果を確認します。
